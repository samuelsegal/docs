# Big O Notation

|     notation | name            |
| -----------: | --------------- |
|         O(1) | constant        |
|    O(log(n)) | logarithmic     |
| O((log(n))c) | polylogarithmic |
|         O(n) | linear          |
|        O(n2) | quadratic       |
|        O(nc) | polynomial      |
|        O(cn) | exponential     |
